Link4Fun

Link4Fun my personal project using Lua + OpenResty and it allows me to create a short URL of the content I'd to share.

Features:

  • Fast URL shortener
  • Markdown editor with syntax highlight support
  • Create markdowns up to 100K
  • Create snippets for multi-languages up to 50K
  • Dark mode according to your browser preferences

Pay attention:

  • All data are public
  • Data can be removed if the content is considered inappropriate by me
